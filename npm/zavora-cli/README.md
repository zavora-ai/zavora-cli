# @zavora-ai/zavora-cli

Install `zavora-cli` from npm with a prebuilt binary from GitHub Releases.

## Install

```bash
npm i -g @zavora-ai/zavora-cli
```

Supported platforms:
- macOS: `x64`, `arm64`
- Linux: `x64`, `arm64`

If your platform is not supported, install from cargo:

```bash
cargo install zavora-cli
```

## Usage

```bash
zavora-cli chat
```
